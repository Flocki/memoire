Description

Qu'est ce que ce site ?

Tatata est un site de recrutement pour les jeunes journalistes fraichement diplômés ainsi que pour les employeurs cherchant du personnel.

Pourquoi ce site ?

Ce site a été créé dans le but d'aider les jeunes diplômés à trouver du travail. En effet, la recherche de travail n'est pas la chose la plus facile en sortant des études, manque d'expérience, peur de se lancer, ... Nous sommes donc là pour aider ces personnes en les mettant en contact avec les employeurs.
